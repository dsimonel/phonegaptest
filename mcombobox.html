MComboBox allows you to create a combobox to select one or multiple values.

  • Control the Theme and color variation of a group of MComboBox with the MobileTheme component associated to them.
  • Change the SystemIcon to a different icon.
  • Change the IconPos value to set where the icon will be placed.
  • Select your own Icon
  • Use Multiselect to allow it to select multiple values.
  • Indicate with isNative that the control will render in the browser's native mode.
  • Add Items that will have the following attributes:
    • Text Text to display
    • Value  Value of the item
    • Group  Indicates the group that the item belongs to.
    • Disabled  Indicates if the itemis disabled
    • PlaceHolder: With this attribute it will become the NULL option of the select to force the user to select an element from the list.
